Privacy Policy

Tik-Tok_Sender is a plugin for the desktop application “Soundboard Portable”. It processes only the data required to authenticate with TikTok (OAuth) and to upload/publish videos on behalf of the user.

Data we process

  • OAuth authorization data (authorization code, access token, refresh token, expiry timestamps)
  • Basic plugin settings (e.g., selected scopes/settings)
  • Video files selected by the user for upload (processed locally and uploaded to TikTok)

Storage

OAuth tokens and plugin settings are stored locally on the user’s device. The plugin does not operate its own external server.

Sharing

The plugin does not sell or share personal data with third parties. Data is sent to TikTok only as required to perform login and uploads.
